## v0.6.0
### Added
- Add a `badgeType` filter to `app.certified.badge.award` GraphQL queries so clients can fetch awards by the referenced badge definition type without joining badge definitions client-side.
### Fixed
- Stop advertising nested-array `any` filters inside another array `any` scope because those filters were accepted by GraphQL validation but could not execute. Nested arrays still expose `isNull` presence checks, while supported single-array `any` filters continue to work.
