## v0.8.0
### Added
- Added `recordTimeline`, a GraphQL query for building creation-time feeds across selected collections. It returns decoded records on `RecordTimelineNode.value`, supports `where.collection.in` and optional `where.did` filters, and uses stable cursor pagination for pages of up to 1,000 records. Existing records are backfilled during migration. Timeline connections intentionally skip exact total counts.
