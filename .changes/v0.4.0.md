## v0.4.0
### Added
- Add certifiedProfileData to public record GraphQL nodes so clients can fetch an author's Certified profile record and its profile-record external labels alongside any indexed record.
- Add a small Prometheus deployment service for scraping Tap metrics in Railway.
### Breaking
- Rename the activity log table to `indexing_activity`. Operators with direct SQL or monitoring against `jetstream_activity` should update those references; delete activity events now store empty or JSON null payloads as `{}` so PostgreSQL JSONB inserts succeed.
### Fixed
- Fix external label hydration for large GraphQL pages so requests for 1,000 records with labels no longer hit SQLite query limits.
- Fix legacy Go backfill CAR record conversion so AT Protocol CBOR values such as CID links, bytes, and blobs are stored with their canonical JSON shapes.
